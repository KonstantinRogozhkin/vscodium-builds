## Технические детали

### Проблема
```powershell
# Неправильно - PowerShell не понимает пробелы в имени файла
.\Code - OSS.exe
```

### Решение
```powershell
# Правильно - использование оператора вызова с кавычками
& ".\Code - OSS.exe"
```
